# TikTok Video Translator & Makeup Style Transfer

This app translates e-commerce TikTok videos from Spanish to English while converting women’s makeup style from Latino to mainstream American.

## Features
1. Automatic speech transcription (Spanish)
2. Translation to English
3. Text-to-speech synthesis in English
4. Face makeup style transfer using BeautyGAN
5. Combines processed frames and new audio into output video

## Quickstart
```bash
pip install -r requirements.txt
python src/process_video.py --input input.mp4 --output output.mp4
```
